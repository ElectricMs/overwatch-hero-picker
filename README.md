# 🎯 守望先锋本命英雄抽取器

一个基于2026年3月最新版本的守望先锋英雄抽取小游戏，支持全部50位英雄！

## 功能特点

- ✅ **50位英雄**：涵盖守望先锋所有英雄（2026年3月最新版本）
- ✅ **角色分类**：重装/输出/支援三大角色类型
- ✅ **随机抽取**：长按按钮抽取你的本命英雄
- ✅ **英雄图鉴**：查看所有英雄及收集状态
- ✅ **详细信息**：点击英雄查看详情（需解锁）
- ✅ **收集进度**：追踪收集进度，显示百分比
- ✅ **本地存储**：刷新页面后数据不会丢失
- ✅ **响应式设计**：完美支持手机和电脑
- ✅ **精美UI**：渐变色设计，流畅动画

## 游戏玩法

1. 长按"抽取英雄"按钮（2秒）
2. 等待抽取动画完成
3. 查看你抽取到的本命英雄
4. 在"英雄图鉴"中查看所有英雄
5. **点击英雄**查看详细信息（已收集的英雄显示完整信息）
6. 收集更多英雄，完善你的英雄图鉴！

## 功能说明

### 英雄图鉴
- 🟢 **绿色**：已收集的英雄，显示完整信息和emoji
- 🔒 **灰色**：未收集的英雄，显示锁定状态和问号
- 点击任意英雄查看详情（已收集英雄显示完整描述）

### 收集进度
- 显示已解锁英雄数量 / 总数
- 进度条显示收集百分比
- 支持重置功能（点击"🔄 重置"按钮）

### 数据存储
- 所有数据保存在浏览器 LocalStorage 中
- 清除浏览器数据会丢失收藏记录
- 可以手动重置收藏记录

## 自定义

### 添加/修改英雄
编辑 `script.js` 中的 `heroes` 对象：

```javascript
const heroes = {
    tank: [
        { name: '英雄名', english: 'English Name', emoji: '🎯', desc: '英雄描述' }
    ],
    // ...
};
```

### 修改颜色
编辑 `style.css` 中的渐变色：

```css
/* 主色调渐变 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 角色颜色 */
.type-tank { background: #e74c3c; }  /* 重装 - 红色 */
.type-damage { background: #f39c12; } /* 输出 - 橙色 */
.type-support { background: #2ecc71; } /* 支援 - 绿色 */
```

### 调整抽取时间
修改 `script.js` 中的 `setTimeout` 时间（默认 2000ms）：

```javascript
setTimeout(() => {
    drawHero();
    // ...
}, 2000);  // 改为你想要的时间（毫秒）
```
